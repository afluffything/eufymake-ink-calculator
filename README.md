# eufyMake E1 — Ink Cost Calculator

A free, browser-based ink cost calculator for the **eufyMake E1 UV printer**. No sign-up, no app, no ads — just open it and start calculating.

🔗 **[Open the calculator](https://yourusername.github.io/eufymake-ink-calculator)**

---

## What it does

- **Per colour mode** — enter exactly how many ml each ink channel uses (Cyan, Magenta, Yellow, Black, White, Gloss, Cleaner) and get an accurate cost per print
- **Total ml average mode** — just enter the total ml used if you don't want to break it down by colour
- **Per item / sticker breakdown** — enter how many items fit on one print sheet and it calculates the ink cost, ml used, and sell price per individual item
- **Sell price calculator** — add your other costs (substrate, labour etc.), set your markup, and instantly see your suggested sell price and profit per item
- **Prints per kit** — tells you how many prints you'll get before any one bottle runs out

## How to use

1. Open the calculator in any browser — works on desktop and mobile
2. Choose your mode: **Per colour** or **Total ml average**
3. Enter the ml used for each ink channel on your print
4. Enter how many items fit on one print sheet
5. Add any other costs (material, time etc.) and adjust the markup slider
6. Read off your suggested sell price and profit per item

## Running locally

No installation needed. Just download `index.html` and open it in any browser — it runs entirely offline with no dependencies.
